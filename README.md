# 📊 Exploratory Data Analysis on Titanic Dataset

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The main goal of the project is to analyze passenger data, identify patterns and trends, and understand the factors that influenced survival during the Titanic disaster.

---

## 🎯 Objectives
- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Handle missing values
- Generate statistical summaries
- Create visualizations for better understanding
- Identify correlations between features
- Extract meaningful insights from the data

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset Information
The dataset used in this project is the Titanic dataset available in Seaborn.

It contains information about:
- Passenger details
- Age
- Gender
- Passenger class
- Ticket fare
- Survival status

---

## 📈 Exploratory Data Analysis Performed

### ✅ Data Cleaning
- Handled missing values
- Removed unnecessary columns
- Checked data types and duplicates

### ✅ Statistical Analysis
- Mean, median, standard deviation
- Summary statistics using `.describe()`

### ✅ Data Visualization
- Survival count plots
- Age distribution histograms
- Passenger class analysis
- Scatter plots
- Heatmaps

### ✅ Correlation Analysis
- Correlation matrix
- Heatmap visualization
- Feature relationship analysis

---

## 🔍 Key Insights
- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers had better survival chances compared to lower classes.
- Passenger class strongly influenced survival.
- Age and fare showed weaker correlations with survival.
